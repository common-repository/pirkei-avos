P.S. Email for shabbes accommodations in Jerusalem.

== Installation ==

This section describes how to install the plugin and get it working.

1. Consult with the relevant halachic authorities
2. Upload `avos.php` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Only after you've been tested with the chance to install this plugin again, and do install it, will you know that you've done a complete installation.
